## Requirements

- **Rust 1.88** or later (managed via `rust-toolchain.toml`)
- **OpenClaw** binary (or a compatible script) for worker task execution

SQLite is automatically bundled at build time — no separate installation required.

## Homebrew (macOS / Linux)

```bash
brew tap co-r-e/tap
brew install starweft
```

シェル補完（bash, zsh, fish）も自動的にインストールされます。

## GitHub Releases

[GitHub Releases ページ](https://github.com/co-r-e/starweft/releases) からビルド済みバイナリをダウンロードできます。

<Tabs items="Linux,macOS,Windows">

<div>

```bash
curl -LO https://github.com/co-r-e/starweft/releases/latest/download/starweft-v0.3.0-x86_64-unknown-linux-gnu.tar.gz
tar xzf starweft-v0.3.0-x86_64-unknown-linux-gnu.tar.gz
sudo mv starweft /usr/local/bin/
```

</div>

<div>

```bash
curl -LO https://github.com/co-r-e/starweft/releases/latest/download/starweft-v0.3.0-aarch64-apple-darwin.tar.gz
tar xzf starweft-v0.3.0-aarch64-apple-darwin.tar.gz
sudo mv starweft /usr/local/bin/
```

</div>

<div>

PowerShell で zip をダウンロードし、`PATH` の通ったディレクトリに配置してください。

</div>

</Tabs>

## cargo install

```bash
cargo install starweft
```

## ソースからビルド

```bash
git clone https://github.com/co-r-e/starweft.git
cd starweft
cargo build --release
```

バイナリは `target/release/starweft` に生成されます。

## Platform Notes

<Tabs items="Linux,macOS,Windows">

<div>
- Default transport: `local_mailbox` (file-based via Unix socket paths)
- Data directory: `~/.starweft`
- Secret key files are protected with `0600` permissions
</div>

<div>
- Default transport: `local_mailbox` (file-based via Unix socket paths)
- Data directory: `~/.starweft`
- Secret key files are protected with `0600` permissions
- Both Intel and Apple Silicon are supported
</div>

<div>
- Default transport: `libp2p` (TCP localhost)
- Data directory: `%LOCALAPPDATA%\starweft`
- Secret key files are protected with read-only + hidden attributes
- Unix socket transport (`local_mailbox`) is **not available** on Windows
</div>

</Tabs>
